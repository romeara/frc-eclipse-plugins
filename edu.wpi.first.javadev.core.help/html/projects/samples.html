Creating a Sample Project

Sample projects are simple to create. Select "New Project" from the toolbar or context menu. Then select any of the available projects from the "FRC Robot Sample" category. the project will be created in your workspace with all required source files. There are five available samples, detailed below.

The Circle Tracker Demo demonstrates using the camera to track a circular target.

The Dashboard Example demonstrates use of Dashboard commands.

Default Code is the stock code for an FRC robot.

The Enhanced I/O sample demonstrates advanced input and output options available to teams in WPILibJ.

The Tracker Demo shows more ways to use the camera to track objects.
